# My_Learning_R

Welcome to my personal workspace for learning, unlearning, and mastering R. 

## About This Repository
Hi, I'm Alule Robert. This repository serves as a living log of my coding journey. I am using this space to document my progress, test scripts, and build reproducible workflows, with a strong focus on bioinformatics, biochemical research, and statistical modeling. 

The goal here is not just to write code, but to write *better* code—breaking old habits and adopting best practices for data science.

## Current Focus Areas
* **Bioinformatics & Transcriptomics:** Processing and analyzing single-cell RNA-seq (scRNA-seq) data.
* **Statistical & Epidemiological Modeling:** Implementing Generalized Additive Models (GAMs), Poisson/Negative Binomial regression, and survival analysis (Kaplan-Meier, Cox Proportional Hazards).
* **Data Visualization:** Creating publication-quality biological and epidemiological plots.
* **Reproducible Research:** Building clean, well-documented analytical pipelines for malaria research data.
